# ProjectNest

A web application that recommends academic books based on course syllabi using machine learning.

## Features

- Student authentication
- Course and subject selection
- Book recommendations based on syllabus matching
- Support for multiple regulations and branches
- Responsive web interface

## Setup

1. Clone the repository
```bash
git clone https://github.com/KadariPavani/Syllabus-Nexus.git
cd ProjectNest
```

2. Create and activate virtual environment
```bash
python -m venv env
source env/bin/activate  # On Windows: env\Scripts\activate
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Run the application
```bash
cd project
python app.py
```

## Technologies Used

- Python Flask
- Machine Learning (scikit-learn)
- HTML/CSS
- JavaScript
